# Electronic-Circuits--Clapp-Oscillator
## AIM:
To simulate a Clapp Oscillator and to obtain the output.
## APPARATUS REQUIRED:
PC with LT Spice software

## SIMULATION PROCEDURE:
1. Open the Ltspice software.<br>
2. FileNew Schematic.<br>
3. Click component symbol  to browse and select the components.<br>
4. Click OK and place the component.<br>
5. Search for resistor from component menu or from the tab icons.<br>
6. Place the resistors.<br>
7. Use ctrl + R to rotate the components.<br>
8. Use wire symbol  to connect the components as per the circuit diagram.<br>
9. Right click each components to give the values.<br>
10. Use run button  to run the simulation.<br>
11. Set the step time and click ok.<br>
12. Output dialog box will open.<br>
13. In the output dialog box right click and select add traces.<br>
14. Node voltages and current of each components names will be displayed.<br>
15. Select the required node voltages (V(n001)) to get the waveforms.<br>
16. To get the voltage across a component the name of the trace should represent both the nodes. example V(n001,n002))
